# Spam Mail Detector

## Objective
To detect spam messages using machine learning and natural language processing.

## Algorithm Used
- Multinomial Naive Bayes

## Steps Performed
- Loaded SMS spam dataset
- Preprocessed text data
- Converted text to numerical features using TF-IDF
- Trained Naive Bayes classifier
- Evaluated model performance

## Result
Achieved high accuracy in identifying spam messages.
